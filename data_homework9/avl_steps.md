# 🌳 AVL 树构建与旋转实操

> **插入序列**: `[30, 20, 10, 25, 40, 35, 50]`
> **约定**: BF = 左子树高 − 右子树高 | 空结点 h = −1 | 叶结点 h = 0

---

## 步骤 1：插入 30

```
          (30)
         ┌─┴─┐
         ●   ●
```

> BF(30) = 0 ✅ 无失衡

**中序**: `[30]`

---

## 步骤 2：插入 20

```
          (30)  BF=+1
          ┌┘
        (20)  BF=0
       ┌─┴─┐
       ●   ●
```

> 20 < 30，插左子树。BF 仍在 [−1,+1] 内 ✅ 无失衡

**中序**: `[20, 30]`

---

## 步骤 3：插入 10 → 🔴 LL 失衡！

```
          (30)  BF=+2  ← 失衡!
          ┌┘
        (20)  BF=+1     ← 旋转轴
        ┌┘
      (10)  BF=0
     ┌─┴─┐
     ●   ●
```

| 项目 | 内容 |
|:-----|:-----|
| **失衡类型** | **LL**（左子树的左子树上插入） |
| **失衡结点** | 30（BF 超出 +1，达到 +2） |
| **失衡路径** | 30 → 20 → 10（同侧左偏） |
| **旋转轴** | **20**（失衡结点左孩子，中间结点） |
| **旋转方式** | **右旋（顺时针）** |

```
        右旋前:                      右旋后:

          30(+2)                      20(0)
          /                           /   \
        20(+1)         ──▶          10(0)  30(0)
        /
      10(0)
```

---

## 步骤 4：LL 右旋后

```
          (20)  BF=0
         ┌─┴──┐
      (10)    (30)
     BF=0     BF=0
    ┌─┴─┐    ┌─┴─┐
    ●   ●    ●   ●
```

> 以 20 为轴右旋：30 下沉为 20 的右孩子。所有 BF 归零 ✅ 平衡恢复

**中序**: `[10, 20, 30]`

---

## 步骤 5：插入 25

```
            (20)  BF=−1
          ┌──┴───┐
        (10)     (30)  BF=+1
       BF=0      ┌┘
      ┌─┴─┐    (25)  BF=0
      ●   ●   ┌─┴─┐
              ●   ●
```

> 25 介于 20~30，插到 30 左子树。各结点 BF 仍在范围内 ✅ 无失衡

**中序**: `[10, 20, 25, 30]`

---

## 步骤 6：插入 40

```
            (20)  BF=−1
          ┌──┴────┐
        (10)      (30)  BF=0
       BF=0     ┌──┴──┐
      ┌─┴─┐   (25)   (40)
      ●   ●  BF=0    BF=0
             ┌─┴─┐  ┌─┴─┐
             ●   ●  ●   ●
```

> 40 > 30，插到 30 右子树。30 的 BF 从 +1 回到 0 ✅ 无失衡

**中序**: `[10, 20, 25, 30, 40]`

---

## 步骤 7：插入 35 → 🔴 RR 失衡！

```
            (20)  BF=−2  ← 失衡!
          ┌──┴────┐
        (10)      (30)  BF=−1     ← 旋转轴
       BF=0     ┌──┴───┐
      ┌─┴─┐   (25)    (40)  BF=+1
      ●   ●  BF=0     ┌┘
             ┌─┴─┐  (35)  BF=0
             ●   ●  ┌─┴─┐
                    ●   ●
```

| 项目 | 内容 |
|:-----|:-----|
| **失衡类型** | **RR**（右子树的右子树上插入） |
| **失衡结点** | 20（BF 超出 −1，达到 −2） |
| **失衡路径** | 20 → 30 → 40 → 35（同侧右偏） |
| **旋转轴** | **30**（失衡结点右孩子，中间结点） |
| **旋转方式** | **左旋（逆时针）** |

```
        左旋前:                         左旋后:

          20(−2)                          30(0)
         /    \                          /    \
      10(0)  30(−1)        ──▶        20(0)  40(+1)
             /   \                    /  \    /
          25(0) 40(+1)             10  25  35
                 /
               35(0)
```

---

## 步骤 8：RR 左旋后

```
               (30)  BF=0
          ┌─────┴──────┐
        (20)          (40)  BF=+1
       BF=0           ┌┘
     ┌──┴──┐        (35)  BF=0
   (10)   (25)     ┌─┴─┐
  BF=0    BF=0     ●   ●
  ┌─┴─┐  ┌─┴─┐
  ●   ●  ●   ●
```

> 以 30 为轴左旋：20 下沉为 30 左孩子，30 上升为新根。
> 40 的 BF 仍为 +1（左子树 35 存在）。其余结点平衡 ✅

**中序**: `[10, 20, 25, 30, 35, 40]`

---

## 步骤 9：插入 50 → ✅ 最终 AVL 树

```
               (30)  BF=0
          ┌─────┴──────┐
        (20)          (40)  BF=0
       BF=0         ┌──┴──┐
     ┌──┴──┐      (35)   (50)
   (10)   (25)   BF=0    BF=0
  BF=0    BF=0  ┌─┴─┐  ┌─┴─┐
  ┌─┴─┐  ┌─┴─┐  ●   ●  ●   ●
  ●   ●  ●   ●
```

> 50 > 40，插到 40 右子树。40 BF 从 +1 回到 0。**所有结点 BF ∈ [−1, 0, +1]** ✅ AVL 树构建完成！

**中序**: `[10, 20, 25, 30, 35, 40, 50]`
