# 6顶点无向带权图 MST 求解

## 邻接矩阵

|   | A | B | C | D | E | F |
|---|---|---|---|---|---|---|
| **A** | 0 | 2 | 0 | 3 | 0 | 0 |
| **B** | 2 | 0 | 4 | 0 | 1 | 0 |
| **C** | 0 | 4 | 0 | 0 | 0 | 5 |
| **D** | 3 | 0 | 0 | 0 | 6 | 0 |
| **E** | 0 | 1 | 0 | 6 | 0 | 2 |
| **F** | 0 | 0 | 5 | 0 | 2 | 0 |

## 边集 (去重, 按权升序)

| 边 | 权重 |
|----|------|
| B–E | 1 |
| A–B | 2 |
| E–F | 2 |
| A–D | 3 |
| B–C | 4 |
| C–F | 5 |
| D–E | 6 |

---

## 一、Prim 算法（从 A 出发）

| 步骤 | 已选顶点 | 候选边 | 选中 |
|------|----------|--------|------|
| 1 | {A} | A–B(2), A–D(3) | **A–B(2)** |
| 2 | {A,B} | A–D(3), B–C(4), **B–E(1)** | **B–E(1)** |
| 3 | {A,B,E} | A–D(3), B–C(4), D–E(6), **E–F(2)** | **E–F(2)** |
| 4 | {A,B,E,F} | **A–D(3)**, B–C(4), C–F(5), D–E(6) | **A–D(3)** |
| 5 | {A,B,D,E,F} | **B–C(4)**, C–F(5), D–E(6) | **B–C(4)** |

**MST 边集**: A–B(2), B–E(1), E–F(2), A–D(3), B–C(4)  
**总权重**: 2 + 1 + 2 + 3 + 4 = **12**

---

## 二、Kruskal 算法

| 步骤 | 边 | 并查集状态 | 操作 |
|------|----|------------|------|
| 1 | B–E(1) | {B,E}, {A}, {C}, {D}, {F} | ✅ 加入 |
| 2 | A–B(2) | {A,B,E}, {C}, {D}, {F} | ✅ 加入 |
| 3 | E–F(2) | {A,B,E,F}, {C}, {D} | ✅ 加入 |
| 4 | A–D(3) | {A,B,D,E,F}, {C} | ✅ 加入 |
| 5 | B–C(4) | {A,B,C,D,E,F} 全部连通 | ✅ 加入 |
| — | C–F(5) | 已连通 | ❌ 跳过 |
| — | D–E(6) | 已连通 | ❌ 跳过 |

**MST 边集**: B–E(1), A–B(2), E–F(2), A–D(3), B–C(4)  
**总权重**: 1 + 2 + 2 + 3 + 4 = **12**

## 结论

Prim 与 Kruskal 结果一致：MST 总权重 **12**，包含边

> **B–E(1), A–B(2), E–F(2), A–D(3), B–C(4)**
